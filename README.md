# Análise de Sinistros (Acidentes) em Rodovias Federais — PRF (2021–2025)

## Visão Geral
O projeto visa reunir dados coletados pela Policia Rodoviária Federal, buscando entender os padrões de risco e os fatores que mais levam a acidentes para responder perguntas como:
- Como se dá a distribuição dos acidentes por UF e causa? 
- Quais dias e horários são mais associados com acidentes?
- Como o clima afeta?
- Como o tipo de pista influencia?

A análise possuirá testes estatísticos para validar os insights aqui encontrados, além de possuir um dashboard interativo construido no Power BI para fácil visualização.

## Dataset
Fonte: Dados Abertos da PRF <br>
Período: 2021 a 2025<br>
Volume: 342.620 registros de ocorrências<br>
Arquivo utilizado: agrupado por ocorrência (não por pessoa envolvida)

## Ferramentas
- Python (Pandas, Scipy, Seaborn);
- Power BI.

## Nota
Esse projeto tem o propósito de aprendizado na área de análise de dados, estatística e Power BI.<br> Além disso, este repositório será a base de um futuro projeto maior, que consistirá no uso dos dados tratados aqui para construir um modelo de machine learning com o intuito de criar um simulador interativo de risco nas rodovias brasileiras.<br> <br>
