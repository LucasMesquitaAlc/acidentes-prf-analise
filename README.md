# Análise de Sinistros (Acidentes) em Rodovias Federais — PRF (2021–2025)

## Visão Geral
O projeto visa reunir dados coletados pela Policia Rodoviária Federal, buscando entender os padrões de risco e os fatores que mais levam a acidentes para responder perguntas como:
- Como se dá a distribuição dos acidentes? 
- Quais Dias e horários são mais correlacionados com sinistros?
- Como o clima afeta?
- Como o tipo de pista influencia?

A análise possuirá testes estatísticos para validar os insights aqui encontrados, além de possuir um dashboard interativo construido no powerbi para fácil visualização.

## Dataset
Fonte: Dados Abertos da PRF
Período: 2021 a 2025
Volume: 342.620 registros de acidentes por ocorrência
Arquivo utilizado: agrupado por ocorrência (não por pessoa envolvida)

## Ferramentas
- Python (Pandas, Scipy, Seaborn);
- Power BI.

## Futuro Projeto
Este repositório será a base de um futuro projeto maior, que consistirá no uso dos dados tratados aqui para construir um modelo de machine learning com o intuito de criar um simulador interativo de risco nas rodovias brasileiras.
